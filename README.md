## About This Project

A general-purpose **Epidemic Spread Prediction System** 
built for the CodeCure AI Hackathon (SPIRIT 2026, IIT-BHU).

While COVID-19 serves as our primary dataset, the system 
is designed to work with any infectious disease data and 
incorporates:

- Outbreak prediction using Facebook Prophet + LSTM
- Hotspot detection using Random Forest classification  
- Transmission modeling using SIR compartmental model
- Effective Reproduction Number (Rt) estimation
- Environmental & demographic risk factors
- Interactive geographic risk mapping
